# N26
Small project following [The Clean Architecture](https://blog.cleancoder.com/uncle-bob/2012/08/13/the-clean-architecture.html)

## Objective
The main objective of the project is to allow the creation of transactions and basic statistics of recent transactions

## Requirements
- Java 8
- Maven

## Used techs
- [Springboot](https://spring.io/projects/spring-boot)
- [JUnit 4](https://junit.org/junit4/)
- [Mockito](https://site.mockito.org/)

## Running the application
```
mvn spring-boot:run
```

## Running tests
```
mvn clean test
```
